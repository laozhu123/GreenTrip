 绿出行
==
## 介绍 ##

能购是一款绿色出行软件，当用户设定目的地后可选择：公交，自行车，步行，混合四种模式，从而达到绿色出行，用户还可进行朋友圈的展示，展示自己的节能记录。接入地图，三方登陆，推送。
 


## 技术##

MVC模式，缓存，路径择优算法。
